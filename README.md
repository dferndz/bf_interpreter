# Brainfuck interpreter

### Compile
* Clone this repo
* Run ```make```   
* Run ```bf <filename>```

Run ```bf -h``` for help.

### Usage
```
Usage: bf [-h] [-oid <filename>] filename
 -h             view this help message
 -i <filename>  stdin file
 -o <filename>  stdout file
 -d <dilename>  dump cleaned brainfuck instructions
 ```
